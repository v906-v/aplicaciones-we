Introduccion

MySQL es un sistema de gestion de bases de datos relacional (RDBMS) de codigo abierto que utiliza el lenguaje SQL (Structured Query Language) para gestionar y manipular datos. Fue desarrollado por MySQL AB, una empresa sueca, y actualmente es propiedad de Oracle Corporation. MySQL es ampliamente utilizado en aplicaciones web y es conocido por su rendimiento, fiabilidad y facilidad de uso.

show database

El comando SHOW DATABASES en MySQL se utiliza para listar todas las bases de datos disponibles en el servidor MySQL al que estás conectado. Este comando es útil para ver qué bases de datos existen y para verificar que tienes acceso a ellas.
